This project is a classic **Tic Tac Toe game** built using **Vanilla JavaScript, HTML, and CSS**, focusing on clean gameplay logic and a smooth user experience. The game allows two players to compete in turns, with real-time updates to the game board and instant detection of win and draw conditions.

The application implements efficient game state management, ensuring accurate move validation and preventing invalid actions. A simple and intuitive user interface keeps the focus on gameplay, while dynamic feedback enhances user interaction.

This project serves as a great demonstration of core JavaScript concepts such as event handling, conditional logic, DOM manipulation, and state control. It is ideal for beginners learning JavaScript as well as for portfolio showcase purposes.
